The **final project submission** is located in: Final_Project/Air_Passenger_Demand_Forecaster


---

## How to run the project (Nuvolos)

From the repository root:

cd Final_Project/Air_Passenger_Demand_Forecaster
conda env create -f environment.yml
conda activate Air_Passenger_Demand_Forecaster
python main.py

If the environment already exists and you want to recreate it:

conda env remove -n Air_Passenger_Demand_Forecaster
conda env create -f environment.yml
